
# TypeMaster

Projeto de Pagina de Produto em Landing Page, utilizando React com JavaScript com TailwindCSS.

## Deploy

O Projeto está em hospedagem na Vercel o link esta abaixo:

Link: https://typemaster-page.vercel.app/

## Instalação

Instalação das dependencias do projeto

```bash
  npm install
```

Rodar o Codigo do Projeto

```
    npm start
```
    
## Aprendizados

O Projeto do TypeMaster foi minha primeira introdução com React, foi um projeto que guiou para criação de componentes com props e tambem tive desenvolvimento com TailwindCSS.
